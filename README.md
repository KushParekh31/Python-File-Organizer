# 📂 File Organizer

Automatically organize files into folders according to their extensions.

## Features

- Organizes Images
- Organizes Videos
- Organizes Music
- Organizes Documents
- Organizes Archives
- Organizes Programs
- Organizes Code
- Unknown files go into Others

## Installation

```bash
git clone https://github.com/YourUsername/File-Organizer.git
cd File-Organizer
```

Run

```bash
python organizer.py
```

Enter the folder path.

Example:

```
Downloads/
    Images/
    Videos/
    Music/
    Documents/
    Code/
    Others/
```
